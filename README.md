## ğŸ’» Usage

### Navigating the Site
1. Use the navigation bar to jump between pages
2. Click recipe cards to view full recipes with instructions
3. Fill out the contact form to send suggestions
4. Use tag filters to browse by category

### Customizing the Site

**Add a New Recipe:**
1. Create a new HTML file (e.g., `new-recipe.html`)
2. Copy structure from existing recipe pages (e.g., `carne-recipe.html`)
3. Update title, image, ingredients, and instructions
4. Link from `recipes.html` and `tags.html`

**Change Colors/Styling:**
- Edit `css/main.css` to customize colors, fonts, and layout
- Variables are defined at the top of the CSS file for easy tweaking

**Add Navigation Links:**
- Update the `<nav>` section in any HTML file to add new pages
